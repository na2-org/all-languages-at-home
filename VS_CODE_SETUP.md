# Visual Studio Code Setup Guide
## For "Alle Talen Thuis" Project (First Time Frontend Work)

This guide is written for you because you are new to web/frontend development and previously had Microsoft Visual Studio instead of VS Code.

---

## Step 1: Install Visual Studio Code

1. Open your web browser.
2. Go to the official website:
   https://code.visualstudio.com/

3. Click the big blue button that says **"Download for Windows"** (or "Download" if it detects your OS).

4. Run the downloaded `.exe` file.

5. During installation:
   - Check the box **"Add to PATH"** (very important)
   - You can leave the other options as default
   - Click **Install**

6. After installation finishes, **open Visual Studio Code**.

---

## Step 2: Open Your Project in VS Code

1. In VS Code, go to the menu: **File → Open Folder**
2. Navigate to this folder:
   ```
   C:\Users\yawen\Documents\Learning\Projects\all-languages-at-home
   ```
3. Select the folder and click **"Select Folder"**

You should now see your files (`index.html`, `app.js`, `styles.css`, `README.md`, etc.) on the left side.

---

## Step 3: Install the Live Server Extension (Critical)

This extension lets you run your website locally with automatic refresh.

1. On the left sidebar in VS Code, click the **Extensions** icon (it looks like 4 squares).
2. In the search box at the top, type exactly:
   ```
   Live Server
   ```
3. Look for the extension with this information:
   - **Name**: Live Server
   - **Publisher**: Ritwick Dey
   - It should have millions of downloads

4. Click the **Install** button.

**Tip**: If you see many results, search instead for:
```
@publisher ritwickdey
```

---

## Step 4: Run Your Project

1. In the left file explorer, **right-click** on the file `index.html`
2. Choose **"Open with Live Server"**

A browser should automatically open showing your app.

- If it doesn't appear, look at the bottom-right corner of VS Code. You should see a button saying **"Go Live"**. Click it.

---

## Step 5: Important Keyboard Shortcuts

- Hard refresh (to see your latest changes): `Ctrl + Shift + R`
- Open Command Palette: `Ctrl + Shift + P`

## Common Problems & Solutions

**Problem**: "I searched 'Live Server' but I see 'Live Preview' by Microsoft at the top"

**Solution**: Use the search `@publisher ritwickdey` instead.

**Problem**: No "Open with Live Server" option appears

**Solution**: 
1. Make sure you installed the extension by **Ritwick Dey**
2. Restart VS Code completely
3. Check the bottom right corner for the "Go Live" button instead

**Problem**: Changes don't appear after editing

**Solution**: In your browser, press `Ctrl + Shift + R` (hard refresh)
